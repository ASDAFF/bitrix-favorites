## Установка

### Composer
```bash
composer require serginhold/bitrix-favorites
```

### Ручная установка
* Создать папку `serginhold.favorites` в папке `/local/modules/` или `/bitrix/modules/`
* Скопировать файлы модуля в папку `serginhold.favorites`

## Требования
* PHP >= 5.5.0
